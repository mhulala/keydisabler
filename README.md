# Key Disabler
This is an AutoHotkey (.ahk) file designed to disable pressing selected keys while active. 

I have lost many games in Counter Strike to accidentally fat-fingering the windows key while holding S and taking a screenshot instead of playing the game. While there are keyboards and programs that aim to prevent this issue, I found using AHK the simplest and universal solution.

## Dependencies
Requires AutoHotkey to be installed, which can be done at https://www.autohotkey.com/

## Installation
Once AHK is installed, just download the .ahk file and save it somewhere safe (I use my documents folder). Running it for the first time will initiate the boot sequence, after which you can set it to automatically run on startup by **right-clicking** the track icon it generates. **Left clicking the icon toggles the disabler.**

## Customization
Pulling up the left click options also lets you set what key to disable. 
*The current version supports the windows key, escape, caps, num, and scoll lock, insert, and pause/break.*
